# Computer-Vision (Workspace Root)

This workspace contains assignments and utilities for basic Computer Vision tasks. The primary assignment in this workspace is the A1 folder which implements AR tag detection and overlay.

Top-level run instructions:

1. `cd A1`
2. Install dependencies: `pip install numpy opencv-python`
3. Run `python main.py` (see `A1/README.md` for task-specific flags)

For camera calibration, use `A1/caliberate_camera.py` and `A1/get_images.py`.
